## 5 流派分类

| 流派 | 视觉气质 | 代表风格 |
|------|---------|---------|
| **信息建筑派** | 理性、数据驱动、克制网格 | S1 Pentagram / S2 Müller-Brockmann |
| **极简主义派** | 秩序、留白、精致排版 | S3 Kenya Hara / S4 Apple HIG |
| **实验先锋派** | 先锋、生成艺术、视觉冲击 | S5 Sagmeister / S6 Zach Lieberman |
| **运动诗学派** | 动感、沉浸、技术美学 | S7 Field.io |
| **东方哲学派** | 温润、诗意、思辨留白 | S8 Takram |

❌ **差异化规则**：推荐 3 个方向时，必须来自 3 个不同流派。

## 风格详情

### S1 · Pentagram (Michael Bierut)
**流派**：信息建筑派
**哲学**：字体即语言，网格即思想
**核心特征**：
- 极度克制的颜色（黑白 + 1 个品牌色）
- 瑞士网格系统的现代演绎
- 字体排印作为主要视觉语言
- 负空间的战略性使用（60%+ 留白）

**提示词 DNA**：
```
极度克制的颜色系统：黑白 + 1个品牌色 HEX
瑞士网格精确数学间距
字体排印作为主要视觉语言
60%+ 留白比例
数据可视化作为装饰而非填充
```

**代表作**：Hillary Clinton 2016 campaign identity、Vizio brand identity
**适合**：专业发布、企业品牌、演讲稿、数据报告

---

### S2 · Müller-Brockmann
**流派**：信息建筑派
**哲学**：网格是信息的秩序
**核心特征**：
- 网格系统数学般精确
- 信息图表化表达
- 蓝 + 红点缀色
- 无装饰，功能即美学

**提示词 DNA**：
```
网格系统精确到像素
蓝 #003087 + 红 #E4002B 点缀
信息图表作为叙事工具
无装饰元素
数学比例驱动布局
```

**代表作**：世界卫生组织视觉系统、SBB 瑞士联邦铁路
**适合**：数据报告、展览设计、书籍排版、信息可视化

---

### S3 · Kenya Hara (原研哉)
**流派**：极简主义派
**哲学**：设计是虚无的容器
**核心特征**：
- 白底主导，内容极度克制
- 赤土橙 / 灰作为底色
- 一图一语，极简叙事
- 东方留白哲学

**提示词 DNA**：
```
白底主导（#FFFFFF 或 #F5F5F0）
赤土橙 #C04A1A 或暖灰 #8B8680 点缀
极度克制的内容密度
一图一语叙事
留白作为设计元素
```

**代表作**：无印良品视觉系统、松屋银座翻新、白酒品牌设计
**适合**：高端品牌、生活方式产品、艺术展览、东方气质项目

---

### S4 · Apple Human Interface
**流派**：极简主义派
**哲学**：技术应该消失，界面应该透明
**核心特征**：
- SF Pro + 毛玻璃 + 超大圆角卡片
- 极轻阴影，微妙层次
- 留白驱散而非内容填满
- 精致图标，线性图标风格

**提示词 DNA**：
```
SF Pro / -apple-system 字体
毛玻璃效果（backdrop-filter: blur）
超大圆角卡片（12-20px radius）
极轻阴影（0 4px 20px rgba(0,0,0,0.08)）
留白 > 内容密度
```

**代表作**：iOS 17、macOS Sonoma、Apple Watch UI
**适合**：消费级 App、移动端、温暖科技产品、AI 助手界面

---

### S5 · Sagmeister & Walsh
**流派**：实验先锋派
**哲学**：好设计让你想亲吻，差设计让你想离婚
**核心特征**：
- 大字排版，文字即图像
- 幽默文案
- 高饱和撞色
- 名人肖像 + 创意图形

**提示词 DNA**：
```
大字排版文字即图像
高饱和撞色（黄 #FFD700 + 粉 #FF6B6B + 蓝 #4ECDC4）
幽默叙事语气
可选名人肖像元素
视觉冲击 > 克制
```

**代表作**：AIGA 海报、Kenzo 视觉系统、Facebook 新闻室设计
**适合**：品牌 campaign、音乐娱乐、创意展示、个性强烈项目

---

### S6 · Zach Lieberman
**流派**：实验先锋派
**哲学**：让屏幕呼吸
**核心特征**：
- 线条艺术 / motion sketch
- 光影实验
- 生成视觉
- 技术美学 + 诗意表达

**提示词 DNA**：
```
线条艺术为主要视觉元素
SVG stroke 动画
光影明暗对比
生成式图形元素
技术感 + 诗意叙事
```

**代表作**：OpenFrameworks 作品、每日动效练习
**适合**：创意技术展示、交互装置、艺术类项目、动效设计

---

### S7 · Field.io
**流派**：运动诗学派
**哲学**：让体验成为记忆
**核心特征**：
- 粒子系统
- 全屏动效
- 沉浸式 3D 视觉
- 叙事驱动的运动设计

**提示词 DNA**：
```
粒子系统或 3D 场景
全屏动效叙事
沉浸式体验设计
WebGL / Three.js 风格
叙事驱动的运动节奏
```

**代表作**：Field.io 官网、Chrome Experiments
**适合**：品牌发布会、互动网站、创意技术展示、沉浸式体验

---

### S8 · Takram (太莫克制)
**流派**：东方哲学派
**哲学**：设计是时间的艺术
**核心特征**：
- 竖排文字可能性
- 书法感 / 等宽字体
- 纸质感材质
- 留白与时间的诗意关系

**提示词 DNA**：
```
等宽或衬线字体（Noto Serif / Source Han）
纸质感背景（texture）
竖排或不对称布局
留白制造呼吸感
时间感的动画节奏（慢 + 缓）
```

**代表作**：Sony 银行 VI、无印良品书籍
**适合**：高端生活方式品牌、文化艺术项目、日式美学、书籍排版

## 融合风格决策树

当用户提到多个风格时，按以下决策树融合：

```
用户想要"理性+温暖" → S1底色 + S4卡片 + S4字体
用户想要"专业+创意" → S2网格 + S5色彩
用户想要"极简+东方" → S3白底 + S8字体
用户想要"科技+诗意" → S7动效 + S8留白
用户想要"Apple+数据" → S4视觉 + S2信息图表
```
